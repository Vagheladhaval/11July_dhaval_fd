Q-2 How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

1. Applying CSS using jQuery

- You can use the .css() method in jQuery to apply CSS styles to elements.

2. Adding and removing classes

- To add a class, you can use the .addClass() method, and to remove a class, you can use the .removeClass() method.

3. jQuery animations

- jQuery provides methods for simple animations. The .animate() method is commonly used to animate CSS properties.
