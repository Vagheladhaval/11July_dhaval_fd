What are the looping structures in JavaScript? Any one Example?

1> for loop :

- Executes a block of code a specified number of times.

- Example :

  <script>
    for (let i = 1; i <= 5; i++) {
        console.log(i);
    }
  </script>

- Output :

  1
  2
  3
  4
  5




2> for-in loop :

- Iterates over the enumerable properties of an object.

- Example :

<script>
const person = { name: 'Alice', age: 30, city: 'New York' };
for (let key in person) {
    console.log(${key}: ${person[key]});
}
</script>

- Output :

  name: Alice
  age: 30
  city: New York




3> for-of loop :

- Iterates over iterable objects (arrays, strings, etc.) to iterate through the values.

- - Example :

<script>
const colors = ['red', 'green', 'blue'];
for (let color of colors) {
    console.log(color);
}
</script>

- Output :

  red
  green
  blue




4> while loop :

- Executes a block of code while a specified condition is true.

- Example :

<script>
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
</script>

- Output :

  1
  2
  3
  4
  5




5> do-while loop :

- Similar to a while loop, but the block of code is executed at least once before the condition is tested.

- Example :

  <script>
    let i = 1;
    do {
      console.log(i);
      i++;
    } while (i <= 5);
  </script>

- Output :

  1
  2
  3
  4
  5
