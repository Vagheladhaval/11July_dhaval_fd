What is Bom vs Dom in JS?

BOM(Browser Object Model)

- The Browser Object Model is a set of objects that allow interaction with the browser itself.

- BOM is not standardized and can vary between different browsers.

DOM(Docunment Object Model)

- The Document Object Model is a programming interface for HTML and XML documents. It represents the page so thatprograms can change the document structure, style, and content.

- DOM is standardized by the World Wide Web Consortium (W3C).
