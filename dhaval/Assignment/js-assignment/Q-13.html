Q-13. What is the function of the delete operator?

Ans -:

delete operator in JavaScript is used to remove a property from an object or to delete an element

Example:

                 <script>
    
                    var studentObject = {
                        name: "Dhaval",
                        std: 12,
                        city: "Rajkot"
                      };
                      
                      // Using delete to remove a property
                      delete myObject.std;
                      
                      console.log(studentObject);
                      // Output: { name: "Dhaval", city: "Rajkot" }
                      
                </script>
