Q-10 How to read and write a file using JavaScipt?

Ans:-

⟹ On the client side, you can't read or write files in JavaScript browsers.

⟹ It has methods for reading and writing files on the file system that are both synchronous and asynchronous.

⟹ The fs.readFile() and rs.writeFile() methods are used to read and write a file using javascript.

⟹ The file is read using the fs.readFile() function, which is an inbuilt method.

⟹ Syntax of readfile and writefile:-

            fs.readFile(file_name, encoding, callback_function)
            fs.writeFile(file_name, data, options, callback)
