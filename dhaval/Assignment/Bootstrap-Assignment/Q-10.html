Q-10 What is a bootstrap card and how would you create one?

Ans: A Bootstrap card is a flexible and extensible content container.

It's a component provided by the Bootstrap framework, which is widely used for building responsive and mobile-first websites.

Add content inside the card elements (header, body, footer) as needed. This could be text, images, buttons, links, etc.
