Q-3 - What are the default Bootstrap text settings?

Ans:- there are several default text settings provided by classes that you can apply to your HTML elements to style text.

  • 1 .text-left: Aligns text to the left.
  • 2 .text-center: Centers text horizontally.
  • 3 .text-right: Aligns text to the right.
  • 4 .text-justify: Justifies text, adjusting the spacing between words to fill the line width.
  • 5 .text-lowercase: Converts text to lowercase.
  • 6 .text-upercase: Converts text to lowercase.
