Q-9 How can one create an alert in Bootstrap?

Ans: In Bootstrap, you can create alerts using the .alert class along with additional contextual classes to define the color and style of the alert.

We use the .alert class to create an alert box.

We also use the .alert-primary class to give the alert a primary color. Bootstrap provides

additional contextual classes such as .alert-success, .alert-info, .alert-warning, and .alert-danger for different colors and styles.

Inside the with the .alert class, we include the content of the alert message.

The role="alert" attribute is added for accessibility purposes, indicating that the serves as an alert to users.
