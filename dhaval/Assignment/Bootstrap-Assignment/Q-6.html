Q-6 What is a Button Group, and what is the class for a basic Button Group?

Ans: In Bootstrap, a button group is a component provided by the framework for creating and styling button groups efficiently.

The basic class for creating a button group in Bootstrap is .btn-group

Button groups in Bootstrap can also include additional styling and features, such as sizing options, dropdown menus, and toggle buttons, allowing for more versatile and interactive button group layouts.
