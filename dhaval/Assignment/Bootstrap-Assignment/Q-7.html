Q-7 How can you use Bootstrap to make thumbnail?

In this example:

We use the .card component to create a card container for each thumbnail.

Inside each .card, we have an element with the class .card-img-top for the thumbnail image.

We use .card-body to contain the content of the thumbnail, including a title, text, and a button.

The styling for the thumbnail size and spacing is applied using custom CSS.
